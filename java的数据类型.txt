java的数据类型有两大类:
一是基本数据类型，一是引用类型

基本数据类型分为：整数类型，浮点类型，字符类型，布尔类型
        整数类型：byte  一个字节
		             short 二个字节
		             int   四个字节
		             long  八个字节
       浮点类型：float 四个字节
		            double 八个字节
       字符类型：char  二个字节，16位unicode编码
       布尔类型：boolean  //与上面7种数据类型不兼容

引用类型：———对象
          数组属于引用类型          
----------------------------------

基本数据类型的变量是存储在虚拟机栈

Java 如何修改工程名/包名

鼠标点击文件,快捷键 ： win + F2 或 Alt + Shift + R


java 字节码是与类一一对应的

一个java文件中，可以有多个类，但是只能有一个public的类，
这个类的名字必须与文件名一致


数组属于引用类型，各种数据类型数组的初始化

局部变量在使用前，必须初始化
